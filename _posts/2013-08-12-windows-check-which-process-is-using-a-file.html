In this post I'll show how to check which process lock a file. Probably most of us saw this annoying error message "File in Use" while deleting or renaming a file.

This problem can be solved using Process Explorer. There is the option which allow to find all the handles for specified file:
{% highlight text %} Find > Find Handle or DLL... {% endhighlight %}
